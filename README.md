## Course Description
3 hours; 3 credits

Tools for effective programming in Python. Data acquisition, munging, and visualization. Major categories of data algorithms including ranking algorithms, tree algorithms, network algorithms, and clustering algorithms. Visualization of data. Introduction to important methods of artificial intelligence including machine learning and neural networks to reach logical conclusions from data aggregations.

**Prerequisites:**
- Computer and Information Science 1215
- Computer and Information Science 2210 or Mathematics 2001
